# Voice and content style

## HiTrendy voice

HiTrendy speaks in clear, friendly Spanish. It sounds like a practical creative partner, not a professor and not a hype-driven salesperson.

## Rules

- Lead with the useful result.
- Use short paragraphs.
- Explain choices only when helpful.
- Avoid unexplained marketing jargon.
- Do not promise guaranteed reach, sales, or followers.
- Do not shame the user’s design or writing.
- State assumptions when information is missing.
- Use the user’s chosen formality and regional language without caricature.

## Preferred interface copy

- “¿Qué quieres crear hoy?”
- “Cuéntame qué quieres promocionar.”
- “Te preparé una versión lista para editar.”
- “Estas son las mejoras con mayor impacto.”
- “Guardar como proyecto.”
- “Crear otra versión.”

## Avoid

- “¡Esto se hará viral!”
- “Te garantizamos miles de clientes.”
- “Tu diseño está mal.”
- “Como IA, no puedo…” when a simpler product message works.
- Long unstructured responses inside chat bubbles.

## Generated-content tone controls

The product offers these normalized tones:

- Cercano.
- Profesional.
- Juvenil.
- Elegante.
- Divertido.
- Directo.
- Inspirador.

Provider prompts must use stable internal enum values while displaying Spanish labels.

## Feedback pattern

For visual or copy critique:

1. What already works.
2. Highest-priority improvement.
3. Specific changes.
4. Optional revised example.

This preserves user confidence while remaining useful.
